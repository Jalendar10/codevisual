# Change Log

## [1.0.0] - 2026-03-20

### Added
- Interactive React Flow codebase visualization
- Language-agnostic parsing: TypeScript, JavaScript, Python, Java, Go, Rust, C#, C/C++, Ruby, PHP, Swift, Kotlin, Scala, Dart, Lua, Shell
- Right-click context menu: "Visualize This File" and "Visualize This Folder"
- AI-powered code analysis with OpenAI GPT-4o/4o-mini and GitHub Copilot
- Multi-provider AI support with automatic fallback
- Git change analysis — analyzes diffs, identifies breaking changes, rates compatibility
- Auto-analysis on git push with comments pushed to git notes
- File, function, method, class, and package dependency mapping
- Test case visualization and test data flow mapping
- Drag-and-drop graph editing
- Multiple layout algorithms: Dagre, Tree, Radial
- Search and filter: by language, symbol type, file name
- Export to JSON, HTML Report, Markdown Report
- VS Code diagnostics integration — AI issues show as editor warnings
- Activity bar panel with Code Structure and AI Analysis views
- Keyboard shortcut: Ctrl+Shift+V / Cmd+Shift+V
- Package detection: npm, pip, Go modules, Cargo, Maven, Gem, pub
- Dark and light theme support (auto-detects VS Code theme)
